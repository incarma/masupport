# django_ma 보안 및 성능 체크리스트 등급 기준 지침서

## 1. 등급 정의

- 하: 영향 미미 / UI 수준
- 중하: 부분 기능 문제 / 성능 저하
- 중: 운영 영향 가능
- 중상: 장애 가능
- 상: 보안/데이터 영향
- 최상: 즉시 사고 가능

---

## 2. 보안 체크리스트

### 최상
- 파일 직접 접근 (.file.url)
- 서버 권한 검증 누락

### 상
- sanitize 미적용
- 업로드 검증 부족
- fail token 보호 부족

### 중상
- audit 로그 누락
- CSRF 미적용

### 중
- 외부 URL 검증 부족
- 권한 스코프 불일치

### 중하
- 상세 에러 노출

### 하
- 로그 마스킹 부족

---

## 3. 성능 체크리스트

### 최상

### 중상

### 하
- CSS scope 누수
