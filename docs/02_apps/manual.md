# manual 앱 기준 문서 (최종 리팩토링)

## 1. 개요

manual 앱은 내부 매뉴얼 CMS로 다음 구조를 가짐: - Manual → Section →
Block → Attachment

------------------------------------------------------------------------

## 2. 보안 기준 (강화)

### 2.1 파일 접근

-   .url 직접 사용 금지
-   반드시 다운로드 view 경유

### 2.2 업로드 정책

-   확장자 화이트리스트
-   MIME 검증
-   size 제한

### 2.3 XSS 방어

-   Quill HTML sanitize
-   unsafe script 제거

### 2.4 권한 정책

-   ensure_superuser_or_403
-   manual_accessible_or_denied

------------------------------------------------------------------------

## 3. JSON 규약

성공: { "ok": true }

실패: { "ok": false, "message": "..." }

------------------------------------------------------------------------

## 4. JS 구조

-   dataset 기반 boot
-   중복 바인딩 방지
-   BFCache 대응

------------------------------------------------------------------------

## 5. 트랜잭션

-   reorder / move atomic 필수

------------------------------------------------------------------------

## 6. 감사 로그

-   첨부 다운로드
-   매뉴얼 CRUD

------------------------------------------------------------------------

## 7. 운영 체크리스트

-   파일 직접 접근 차단
-   업로드 검증 적용
-   XSS 방어 적용
-   권한 검증 완료
